# vue-ecommerce
Vue電商網站，Vue CLI 3建立開發環境，Vuex儲存狀態容器，串接由Node.js Express MongoDB Atlas Heroku建立之RESTful API取得資料，Vue Router管理路由，localStorage儲存購物車資訊，內建Service Worker製作Progressive Web App。
